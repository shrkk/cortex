<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| NSEvent global monitor → notch view | Keyboard events; only fires when notch window is key or monitor is global |
| Clipboard → CortexIngest.handleClipboard | Clipboard contents are user-controlled |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-07-01 | Elevation of Privilege | NSEvent global keyboard monitor | mitigate | AXIsProcessTrustedWithOptions checked; monitor only active when CortexSettings.enabled = true; ⌘V is non-destructive (read-only from clipboard) |
| T-02-07-02 | Tampering | surgical edit to drop handler | mitigate | #if CORTEX_ENABLED guard preserves original behavior when guard is false; originalHandleDrop extracted verbatim |
| T-02-07-03 | Information Disclosure | Clipboard contents sent to backend | accept | User explicitly triggers ⌘V while the notch is active; intentional action; content is user's own clipboard data |
| T-02-07-04 | Spoofing | CortexCourseTab pending continuation | accept | continuation is local state; not exposed to external callers; worst case: continuation never resolves (upload stalls, not crashes) |
</threat_model>
